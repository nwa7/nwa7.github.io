Shogureign

  • Catégorie: Application mobile, Web, Serious Game
  • Techniques: React Native
  • Équipe: Pauline GOBE, Guy LUONG, Elia ROARD, Laurence WANG
  • Date de projet: janvier 2023
  • Rôle: Développement du Game Design, Dictionnaire de Kanjis et Menu. Participation à l'élaboration du scénario
  • Lien: Github du projet

Serious game sur l'apprentissage des kanjis

Réalisé dans le cadre d'un projet tutoré par Jérôme Hernandez. Le principe était de réaliser une application mobile serious game reprenant le principe des jeux Tinder-like (Type Reigns) L'apprentissage se fait par le biais d'une boucle histoire formée de cartes aléatoirement tirées où les Kanjis apparaissent. Il est renforcé à l'apparition de cartes Leçons et de petits tests. Le joueur peut avoir accès à sa progression à tout moment depuis le menu dictionnaire qui récapitule les kanjis qu'il a rencontré. Première version du jeu terminée, un peu courte pour être testée mais nous esperons implémenter davantage d'histoire et faire des tests d'ici Juin 2023.
